Events

Įvykiai - puslapyje vykstantys kažkokie veiksmai/įvykiai (pelės paspaudimas, klaviatūros paspaudimas, informacijos pažymėjimas, scroll'inimas, formos pateikimas, failo įkėlimas/atsisiuntimas...).
Norėdami klausyti ar nutinka koks įvykis, turime pradžioje nurodyti (selektinti) kurioje DOM dalyje tas įvykis gali nutikti.
Nusirodžius (selektinus), kurioje dalyje klausomės įvykio - reikia aprašyti įvykio klausymąsį. Tai padarome rašydami "addEventListener" metodą, kuris priima 2 parametrus: koksĮvykis, kąDarytiNutikusĮvykiui.

Document Object Model

Dokumentas - puslapis, kuriame yra visa vartotojui atvaizduojama informacija (tekstas, paveiksliukas, forma, lentelė...).
Objektas - kažkoks puslapio elementas (nuo html tag'o, iki ikonos ar span tag'o).
Modelis - objektų struktūra kažkuriame elemente (tekstas, paveiksliukas, forma, lentelė...).

Norėdami per JS selektinti kažkurią DOM dalį, turime rašyti tokį kodą: "document.querySelector" arba "document.querySelectorAll". "QuerySelector" priima 1 parametrą - selektorių (tokį kokį rašėme CSS'e).
