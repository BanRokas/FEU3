Git ir GitHub

Git - kodo versijavimo programa.

GitHub - svetainė, kurioje keliamos kodo versijos.

  1. Įsirašyti Git'ą
  2. Patikrinti ar git'as įrašytas terminale parašius "git --version"
  3. Susikurti gitHub paskyrą
  4. Config'inti git'ą
    1. git config --global user.name gitHubVartotojoVardas
    2. git config --global user.email gitHubVartotojoEmail
    3. git config --global core.editor code
    4. git config --global core.autocrlf true | input (true jeigu windows | input jeigu mac/linux)
  5. Patikrinti ar tvarkingas config failas - git config --global --edit
  6. Source Control dalyje paspausti Publish to GitHub
